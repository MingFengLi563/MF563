# 2026-03-22 - 微信插件安装与配置

## 微信插件安装

### 安装命令
```bash
npx -y @tencent-weixin/openclaw-weixin-cli@latest install
```

### 插件信息
- **插件名称**: openclaw-weixin
- **版本**: 1.0.2
- **仓库**: https://github.com/BytePioneer-AI/openclaw-china

### 二维码生成
- 微信 API 端点：`https://ilinkai.weixin.qq.com/ilink/bot/get_bot_qrcode?bot_type=3`
- 二维码图片保存位置：`/home/admin/.openclaw/workspace/weixin-qr.png`
- 使用 `chafa` 可在终端显示彩色二维码图片
- 使用 `npx qrcode` 可生成 PNG 图片文件

### 登录流程
1. 运行 `openclaw channels login --channel openclaw-weixin`
2. 扫描二维码
3. 在微信上确认授权
4. 重启网关：`openclaw gateway restart`

### 注意事项
- ASCII 二维码在终端可能显示错位，建议使用图片格式
- 终端图片显示工具：`chafa` (已安装)
- 图片查看器：`eog` (已安装，但服务器无图形界面)

### 文件访问方法
用户访问服务器图片文件的方式：
1. **SCP 下载**（推荐）：`scp admin@服务器 IP:/home/admin/.openclaw/workspace/weixin-qr.png ~/Downloads/`
2. **Base64 编码**：`base64 /home/admin/.openclaw/workspace/weixin-qr.png`
3. **HTTP 临时下载**：`python3 -m http.server 8080` 然后浏览器访问
4. **Zmodem**：`sz /home/admin/.openclaw/workspace/weixin-qr.png` (需 SSH 客户端支持)
