### 5. WhatsApp GIF 发送功能调研

**用户需求**: 查找 Meta WhatsApp 发送 GIF 的官方 API 文档

**关键发现**:

| 渠道 | GIF 支持方式 |
|------|--------------|
| **手机端** | ✅ 支持两种来源：<br>1. 官方动态贴纸库<br>2. 相册视频 (<6 秒) 转 GIF |
| **Cloud API** | ❌ 不支持 .gif 直接发送<br>✅ 需转换为 MP4，以视频消息发送 |

**API 实现方案**:
```json
{
  "type": "video",
  "video": {
    "link": "https://example.com/animation.mp4",
    "caption": "动态内容"
  }
}
```

**技术限制**:
- 格式：必须 MP4（不支持 .gif）
- 大小：≤ 5MB（API）/ ≤ 16MB（视频消息）
- 时长：建议 ≤ 30 秒
- 编码：H.264 + AAC

**产品方案**:
1. **方案 A**（推荐）: 视频消息 + 自动播放（用户体验接近 GIF）
2. **方案 B**: 动态贴纸（需预先创建贴纸包，API 支持有限）
3. **方案 C**: 后端转换 GIF → MP4（需 FFmpeg 服务）

**最新动态**（2026 年）:
- WhatsApp 将从 Tenor 切换到 **Klipy** 作为 GIF 提供商
- 切换时间：2026 年 7 月 1 日
- 原因：Tenor API 将于 2026 年 6 月 30 日停止服务

### 6. GIF 转 WebP 工具调研

**用户需求**: 寻找免费工具将 .gif 转换为 .webp 格式

**推荐工具**:

| 类型 | 工具 | 网址/命令 | 特点 |
|------|------|-----------|------|
| **在线** | Ezgif | https://ezgif.com/gif-to-webp | ✅ 免费、批量、无需注册 |
| **在线** | CloudConvert | https://cloudconvert.com/gif-to-webp | ✅ 高质量、API 可用 |
| **命令行** | FFmpeg | `ffmpeg -i input.gif output.webp` | ✅ 本地批量、产品集成 |
| **命令行** | gif2webp | `gif2webp input.gif -o output.webp` | ✅ Google 官方工具 |
| **Node.js** | sharp | `sharp('input.gif').webp().toFile()` | ✅ 产品集成推荐 |

**产品集成方案**:
- **前端**: canvas + WebP 编码（浏览器原生支持）
- **后端**: FFmpeg 或 sharp 库（Node.js）
- **云服务**: Cloudinary / Imgix（自动转换 + CDN）

**注意事项**:
- WebP 支持动画，但部分工具可能丢失帧
- WebP 文件大小通常比 GIF 小 25-35%
- ⚠️ WhatsApp API 不支持 WebP 直接发送，仍需转 MP4

1. systemd 服务文件是旧版本配置 (`v2026.3.3`)
2. pnpm 全局包的前端资源是 3 月 16 日的旧版本
3. Gateway 从 pnpm 全局目录加载前端资源

**修复步骤**:
1. 更新 systemd 服务文件：`~/.config/systemd/user/openclaw-gateway.service`
   - 版本号从 `v2026.3.3` 更新为 `v2026.3.13`
2. 复制新前端资源：
   - 从 `/opt/openclaw/dist/control-ui/`

3. 重启 Gateway 服务：`systemctl --user restart openclaw-gateway`
