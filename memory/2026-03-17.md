## 23:33 - 数据持久化问题

### 用户反馈
- 版本号没有更新（浏览器缓存问题）
- 数据被清空（localStorage 丢失）

### 原因分析
**版本号问题：**
- GitHub Pages CDN 缓存
- 浏览器缓存 build-info.json
- 已修复：添加双重缓存破坏参数

**数据丢失问题：**
- localStorage 是浏览器本地存储
- GitHub 部署不会清空数据
- 可能原因：
  - 用户清除了浏览器缓存
  - 使用了无痕模式
  - 切换了设备/浏览器
  - 域名/路径变化

## 技术要点记录

### localStorage 特性
- 存储限制：约 5-10MB
- 生命周期：永久（除非手动清除）
- 作用域：同域名同路径
- 不随服务器更新丢失

### GitHub Pages 缓存
- CDN 缓存时间：1-5 分钟
- 强制刷新：Ctrl+Shift+R / Cmd+Shift+R
- 缓存破坏：添加查询参数 `?t=timestamp&v=random`
