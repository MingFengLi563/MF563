### 间距问题修复

**时间**: 17:09-17:15

**根本问题**: HTML 结构混乱，`margin-top` 放错了位置

**错误结构**:
```html
<section id="memory">
  <div class="card" style="margin-top: 15px;">  ← margin 在 card 上
```

**正确结构**:
```html
<section id="memory" style="margin-top: 15px;">  ← margin 在 section 上
  <div class="card">
```

**统一间距标准**: 所有模块之间的垂直间距统一为 **15px**

**修复模块**:
| 模块 | 修复方式 |
|------|----------|
| 记忆系统 | `<section id="memory" style="margin-top: 15px;">` |
| 代理配置 + 飞书集成 | `<div class="grid" style="margin-top: 15px;">` |
| 历史消耗趋势 | `<section id="history" style="margin-top: 15px;">` |
| 记忆文件增长 | `<section id="memory-growth" style="margin-top: 15px;">` |

**Grid 间距**:
```css
.grid {
    gap: 15px;           /* 卡片之间间距 */
    margin-bottom: 15px; /* 与下方模块间距 */
}
```
