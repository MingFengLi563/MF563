## 📦 技能安装三渠道

**优先级顺序**：

1. **ClawHub** — 官方技能市场，一键安装
   - 命令：`clawhub search <关键词>` / `clawhub install <skill-slug>`
   - 网址：https://clawhub.ai
   - 特点：官方认证、稳定可靠

2. **GitHub** — 开源技能仓库
   - 仓库：https://github.com/openclaw/skills
   - 方式：手动克隆或等待 ClawHub 收录
   - 特点：最新技能、可贡献

3. **findskill** — 技能发现工具
   - 用途：发现/搜索技能
   - 特点：辅助查找
