## 核心功能

### 一、聊天管理（聊天）

#### 功能入口
左侧导航栏 → 聊天

#### 主要功能
1. **会话列表**
   - 筛选器：全部、未读、未回复、私聊、群聊、已静音
   - 搜索功能：快速查找特定会话
   - 成员筛选：全部成员、未分配、机器人
   - 会话状态：进行中/历史会话切换

2. **会话操作**
   - 接入会话：点击会话进入聊天界面
   - 结束会话：完成服务后手动结束
   - 分配会话：将会话分配给特定成员
   - 标记静音：屏蔽特定会话通知

3. **聊天界面**
   - 消息发送：支持文本、图片、文件等
   - 快捷回复：使用预设话术快速响应
   - 客户信息：查看客户详细资料和历史记录
   - 机器人托管：启用 AI 机器人自动回复

### 二、客户管理（客户）

#### 功能入口
左侧导航栏 → 客户 → 客户列表

#### 主要功能
1. **客户列表**
   - 显示所有客户信息，包括：
     - ONE ID、社媒 ID、渠道来源
     - 账号信息、分配成员
     - 邮箱、电话、国家、城市、语言
     - 标签、状态、备注
     - 创建时间、最后沟通时间
   - 支持自定义显示字段

2. **客户操作**
   - **添加客户**: 手动创建客户记录
   - **批量操作**: 批量分配、打标签、导出等
   - **导出客户**: 导出为 Excel/CSV 格式
   - **自定义受众**: 创建特定客户分组
   - **字段设置**: 自定义客户信息字段

3. **筛选与视图**
   - 按创建时间筛选
   - 按姓名、邮箱、电话搜索
   - 按标签筛选
   - 视图切换：全部客户、我的客户

#### 会话管理
- **入口**: 客户 → 会话管理
- 管理所有客户会话记录
- 支持批量操作和筛选

#### 标签管理
- **入口**: 客户 → 标签管理
- 创建、编辑、删除客户标签
- 用于客户分类和精准营销

#### 话术库
- **入口**: 客户 → 话术库
- 管理常用回复模板
- 支持分类管理和快速插入

#### 素材库
- **入口**: 客户 → 素材库
- 管理图片、文件等营销素材
- 支持分类存储和快速调用

### 三、营销功能（营销）

#### 1. 消息群发
**入口**: 左侧导航栏 → 营销 → 消息群发

**功能说明**:
- 创建批量发送计划
- 支持按条件筛选目标客户
- 设置发送规则和时间

- 发送成功人数及成功率

3. 设置目标受众
4. 编辑发送内容

## 常见问题

### Q1: 如何添加新的社媒渠道？
1. 进入【集成】页面
2. 在社媒账号列表中选择要添加的渠道
3. 按照指引完成授权连接
4. 配置渠道相关设置

### Q2: 如何设置自动回复？
1. 进入【机器人】→【AI 机器人】
2. 创建或编辑机器人代理
3. 配置自动回复规则和提示词
4. 在【设置】→【项目管理】中启用机器人接待

### Q3: 如何批量发送营销消息？
1. 进入【营销】→【消息群发】
2. 点击"创建计划"
3. 选择目标客户群体
4. 编辑发送内容
5. 设置发送时间和规则
6. 确认并启动计划

### Q4: 如何查看客户来源分析？
1. 进入【报告】→【客户分析】
2. 选择时间范围
3. 添加维度（如社媒渠道、接待成员等）
4. 查看数据报表或导出分析

### Q5: 如何安装聊天插件到网站？
1. 进入【集成】→【聊天插件】
2. 找到对应插件，点击"复制"安装代码
3. 将代码粘贴到网站 HTML 的`</body>`标签前
4. 在网站上测试插件是否正常显示

### Q6: 会话如何自动分配给团队成员？
1. 进入【设置】→【项目管理】→【聊天设置】
2. 开启"会话自动分配"
3. 系统会将未分配会话自动分配给在线且未达上限的成员

### Q7: 如何设置超时未回复提醒？
1. 进入【设置】→【项目管理】→【聊天设置】
2. 找到"超时未回复通知"
3. 设置超时时间（如 5 分钟）
4. 选择提醒方式（标红、邮件等）

### Q8: 无权限的功能如何处理？
- 部分高级功能可能需要特定权限或套餐版本
- 联系管理员升级套餐或开通权限
- 系统会自动跳过无权限功能并记录

## 最佳实践

### 1. 客户管理最佳实践

#### 客户标签体系
- 建立清晰的标签分类（如：意向度、来源、行业等）
- 定期清理和更新标签
- 利用标签进行精准营销

#### 客户跟进策略
- 及时响应新客户咨询（建议 5 分钟内）
- 使用话术库保持回复一致性
- 记录重要沟通内容到备注
- 定期回访高价值客户

### 2. 营销自动化最佳实践

#### 群发策略
- 细分目标受众，避免盲目群发
- 测试不同发送时间，找到最佳时机
- 监控发送效果指标（打开率、回复率）
- 避免频繁发送造成骚扰

#### 模板管理
- 创建多套模板应对不同场景
- 定期更新模板内容
- 使用变量提升个性化程度
- A/B 测试优化模板效果

### 3. 机器人配置最佳实践

#### AI 机器人设置
- 配置清晰的机器人人设和语气
- 提供充分的训练数据和提示词
- 设置合理的人工接管条件
- 定期优化机器人回答质量

#### 自动化流程
- 识别重复性工作并自动化
- 设置清晰的触发条件
- 监控自动化执行效果
- 保留人工干预的灵活性

### 4. 团队协作最佳实践

#### 成员管理
- 合理分配客户资源
- 设置明确的接待上限
- 定期查看成员绩效数据
- 组织培训和经验分享

#### 通知设置
- 根据工作习惯配置通知方式
- 避免通知过载影响效率
- 重要客户设置特别提醒
- 下班时间合理关闭通知

### 5. 数据分析最佳实践

#### 日常监控
- 每日查看新增客户数
- 关注响应时间和解决率
- 分析客户来源渠道质量
- 识别异常数据及时排查

#### 定期复盘
- 每周/每月生成数据报告
- 分析趋势变化找原因
- 基于数据优化运营策略
- 设定改进目标并追踪

### 6. 系统集成最佳实践

#### 渠道管理
- 优先集成高价值渠道
- 统一品牌形象和话术
- 定期检查渠道连接状态
- 及时更新渠道配置

#### 数据同步
- 确保客户数据一致性
- 定期备份重要数据
- 利用 API 实现系统互联
- 注意数据隐私和合规

### 7. 安全与合规

#### 账号安全
- 使用强密码并定期更换
- 启用双因素认证（如支持）
- 定期检查登录设备
- 离职成员及时移除权限

#### 数据合规
- 遵守数据保护法规
- 获取客户同意后再营销
- 提供退订选项
- 妥善保存沟通记录
